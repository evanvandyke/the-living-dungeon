# Inter-Agent Communication Log

## Protocol
- **Builder**: Writes what was built, what's next, blockers.
- **Reviewer**: Writes code feedback, bug reports, design suggestions.
- **QA**: Writes test results, build status, regression alerts.
- **Human**: Can write directives at any time. Agents treat human entries as highest priority.
- Keep entries concise. Tag with your role and iteration/timestamp.
- Read the entire log before writing. Don't repeat what's already been said.
